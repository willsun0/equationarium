# Equationarium — Final Project Report

## Concept

Equationarium asks a playful question: what if an equation were not merely graphed, but raised like a living creature? The result is a game about discovering mathematical organisms, developing their abilities, building stronger bloodlines, and completing a long-term Ascension objective.

## What was built

The application combines a procedural canvas simulation, a creature-raising game, a research progression system, and turn-based combat. Ten adjustable genome variables drive each organism's symmetry, wave frequency, amplitude, phase, instability, pulse, membrane complexity, drift, trail, and color. Every control affects the creature's appearance and contributes positively to its base charge; maximizing every gene activates Omega Overdrive.

The Equation Atlas expands the visual engine beyond a sine wave. It includes basic Euclidean geometry as well as trigonometric, polar, parametric, roulette, topological, statistical, transcendental, sequence-based, and fractal morphologies. Twenty-three forms are available. Every form includes an interactive field-guide page plotting the underlying function and explaining why its parameters produce the organism’s anatomy. All ten genome controls also include mathematical and biological explanations directly beside the sliders.

Mathematics also governs development. A Sequence Reactor lets the player encode Fibonacci, prime, Pascal, Collatz, or harmonic patterns for different stat specializations. The Theorem Matrix turns calculus, vector fields, prime numbers, the golden ratio, chaos theory, fractals, and Euler's identity into permanent creature upgrades.

Habitat tools are purposeful: nutrients train vitality, light trains speed, gravity trains defense, and chaos trains power. All experiments strengthen the bond between player and creature. Preserved specimens retain their entire mathematical development and can be restored, used in battle, or selected as parents. Offspring inherit genes, formula type, sequence DNA, some training, and theorem levels.

The Genome Arena translates the organism's equation, genome charge, training, bond, upgrades, and Math Luck into combat statistics. Battles use strategic actions plus controlled randomness. Victories build streaks and accelerate a reward pass that returns energy, XP, and cosmetic auras to the main progression loop.

The final objective is the Ascension Protocol. It requires bonding, strength, battle victories, completed contracts, breeding, and saved Quantum Energy, tying the major systems into one clear purpose.

## AI-assisted development

ChatGPT was used to plan the game systems, generate and refine the React/TypeScript implementation, design the responsive interface, balance the formulas, diagnose issues, and prepare the final documentation. The completed application was checked with linting, a production build, artifact validation, and rendered-output tests.

## Technical approach

- React 19 and TypeScript
- HTML Canvas for procedural animation
- Vite/Vinext production pipeline
- CSS-only interface art, effects, and responsive layout
- Browser localStorage for persistent progress
- No external media, API keys, or backend database required

## Result

The final project is a self-contained game in which mathematical ideas are not decoration: equations determine anatomy, number sequences determine specialization, theorem upgrades determine development, and mathematical resonance helps determine battle outcomes.
